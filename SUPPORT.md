# Support

## Where to ask questions

If you need help or have questions about how to use `puppet-stunnel`, please:

1. Search the [Issue Tracker](https://github.com/philippeganz/puppet-stunnel/issues) to see if someone else has already asked the same question.
2. Open a new issue using the provided templates.

## Reporting Bugs

If you encounter issues, please first verify whether the issue is related to the **Puppet module logic** or the **underlying Stunnel software**.

If you believe the issue is within `puppet-stunnel`:

- Open a Bug Report using the Issue Tracker.
- Provide the full Puppet agent output with `--debug`.
- Provide your OS version, Puppet version, and Stunnel version.

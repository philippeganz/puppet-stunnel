# Security Policy

## Supported Versions

We provide security updates and patches exclusively for the **latest tagged release** of this module on the Puppet Forge. 

| Version | Supported          |
| ------- | ------------------ |
| Latest Tag | :white_check_mark: |
| Older Tags | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please **open a new GitHub Issue** and apply the `security` label. 

Please note that this is an open-source, community-driven project maintained by an individual. There is no dedicated security team, SLA, or private disclosure bounty program, and issues will generally be tracked and addressed openly in the issue tracker.

*If you wish to privately disclose a critical vulnerability before it is made public, you may use GitHub's native "Report a vulnerability" feature located under the repository's **Security** tab.*
